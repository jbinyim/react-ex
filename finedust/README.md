# 📕 개발이야기
사용자에게 전국의 미세먼지 정보를 제공하고, 사용자가 특정 지역을 클릭하면 해당 지역의 상세 정보를 보여주는 기능을 구현하고 있습니다. fetch를 통해 API 호출을 수행하고, useState와 useEffect를 활용하여 상태를 관리하고 있습니다. react-bootstrap을 사용하여 스타일링을 간편하게 처리한 점도 특징입니다.
